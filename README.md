# Windows下WNMP启动相关脚本
    windows+nginx+memcache+php+redis

## 安装步骤
1. 下载wnmp到电脑上；
2. 在文件夹下安装nginx、php、redis、memcache；
3. 分别配置nginx、php；

## 文件列表
- README.md 说明文件
- startService.bat 启动服务
- stopService.bat 停止服务
- restartService.bat 重启服务
- RunHiddenConsole.exe  Windows下隐藏DOS命令行窗口的程序
